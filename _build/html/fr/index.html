A propos de la documentation de ZOO¶

La documentation ZOO est un processus collaboratif; dès que des utilisateurs contribuent à la documentation sur le Wiki de ZOO, le comité de documentation passe en revue les contributions pour les ajouter à la documentation officielle. Le comité est composé actuellement de:

Nicolas Bozon (nicolas.bozon at gmail.com)
Gérald Fenoy (gerald.fenoy at geolabs.fr)
Jeff McKenna (jmckenna at gatewaygeomatics.com)

Le comité tient à remercier les premiers contributeurs à la documentation: Luca Delucchi et René-Luc D'Hont.
